FOUNDATION F1: SYSTEMS

Tier 1 – Quick Reference

  1. Stack variables properly.
  2. Choose a target variable.
  3. Multiply to create opposites (e.g., 2x and −2x).
  4. Add straight down.
  5. Back-solve.

Tier 2 – Deep Mastery

If A = B and C = D, then A + C = B + D.

We add equality to equality. Elimination works because we strategically force one variable to cancel.

Tier 3 – Tutor Notes

Force agency: “What kills 2?”

If answers are messy fractions, check the ADD step first. Sign errors happen there.

FOUNDATION F2: LITERAL EQUATIONS

Tier 1 – Quick Reference (SADMEP)

  1. Circle the target variable (VIP).
  2. Subtract/Add distant terms.
  3. Divide/Multiply coefficients.
  4. Undo exponents last.

Fraction Rule: Multiply everything by the denominator immediately.

Tier 2 – Deep Mastery

Literal equations are re-packaging. You are not changing truth — only the subject of the sentence.

Undo operations in reverse order.

  • Addition undoes subtraction.
  • Multiplication undoes division.
  • Squaring undoes square root.

Tier 3 – Tutor Notes

“Ugly is fine. Advanced math answers are expressions, not numbers.”

FOUNDATION F3: RADICALS & COMPLEX NUMBERS

Tier 1 – Quick Reference

  • √25 = 5 (Real)
  • √−25 = 5i (Imaginary)
  • i² = −1

Tier 2 – Deep Mastery

√25 = √(5 × 5). A pair escapes the radical.

No real number squares to −1. So we define i = √−1.

Tier 3 – Tutor Notes

Force verbal distinction: “Root 81” vs “81 squared.”

When simplifying complex expressions, treat i like a variable — but remember i² = −1.
